# Documentation

### Install the docs

Clone/Fork the repo and run the following command
```bash
npm install
```

### Run the docs

English
```bash
npm start
```

Japanese
```bash
npm run start:ja
```

### Generate translation strings

English
```bash
npm run write-translations
```

Japanese
```bash
npm run write-translations:ja
```

### Build the docs

```bash
npm run build
```
