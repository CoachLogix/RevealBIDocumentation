# システム要件

## ASP.NET
- ASP.NET Core 3.1 またはそれ以降

## Java
- Java SDK 11.0.10 およびそれ以降
- Maven 3.6.3 およびそれ以降

## Node
- Nodejs 16.3 およびそれ以降

:::info

Mac M1/M2 で開発する場合は、Rosetta をインストールする必要があります。

Rosetta を手動でインストールするには、ターミナルで次のコマンドを実行します:
```bash
/usr/sbin/softwareupdate --install-rosetta --agree-to-license
```

:::
